\# API Testing



This folder contains the Postman files used to test the Swagger Petstore Users endpoint.



\# Files

\- `User API Tests.postman\_collection.json`

\- `New Environment.postman\_environment.json`



\# Scope

Only the \*\*Users endpoint\*\* was tested, as required in the assessment.



\# Prerequisites

\- Postman installed

\- Internet connection

\- Imported Postman collection

\- Imported Postman environment



\# How to Run

1\. Open Postman

2\. Import `User API Tests.postman\_collection.json`

3\. Import `New Environment.postman\_environment.json`

4\. Select the imported environment

5\. Run the requests individually or with the Collection Runner



\# Observations

\- The Swagger Petstore Users endpoint supports basic user creation, retrieval, update, login, logout, and delete operations

\- Both positive and negative scenarios can be validated through Postman tests



\# Blockers

\- No major blocker prevented execution

\- Temporary API response issues or internet instability may affect results



\# Notes

\- Only the Users endpoint was tested, in line with the assessment instruction

\- Tests were written and executed locally before being pushed to GitHub
